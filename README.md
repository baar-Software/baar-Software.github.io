# Baar-Verlag Software Team Website

This repository powers the GitHub Pages website for the Baar-Verlag Software
Team:

https://baar-Software.github.io

It is a static site with no build step. Edit `index.html`, `openinstall/`, and
`styles.css`, then push to `main`.

## Local Preview

Any static file server works:

```powershell
python -m http.server 8080
```

Then open:

```text
http://127.0.0.1:8080
```
