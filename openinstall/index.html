OpenInstall

Install Windows apps through verified packages, not mystery setup files.

OpenInstall is an open source Windows package manager and mini app store for signed `.oip` app packages. It resolves packages, verifies publisher signatures and file hashes, checks policy, asks for consent, and then installs files per-user.

Security model

Verify first. Ask clearly. Install only what was checked.

No arbitrary setup launch

Native `.oip` packages install verified files directly instead of downloading and running random `setup.exe` installers.

Signed package metadata

Publisher keys, package signatures, per-file hashes, and policy checks form the install gate.

User-controlled consent

Resolving a link is side-effect-free. Installation requires an explicit user action after verification.
